Comment \u00e7a marche ?<\/h3>\n\n\n\n

La Fontaine Ysio est un osmoseur con\u00e7u pour filtrer l’eau du r\u00e9seau et qui se pose sur l\u2019\u00e9vier.<\/p>\n\n\n\n

Deux modules de filtration :<\/p>\n\n\n\n

  • Un pr\u00e9filtre<\/strong> \u00e0 charbon pour les grosses mol\u00e9cules : boues, sable, poussi\u00e8re… et aussi le chlore et les odeurs. Il se change tous les 3 \u00e0 6 mois selon notre eau.<\/li>
  • Une membrane osmotique<\/strong> qui filtre \u00e0 0,0001 \u00b5m (microm\u00e8tres) pour les particules fines<\/em> comme m\u00e9taux lourds (aluminium, plomb…), nitrates, r\u00e9sidus m\u00e9dicamenteux, calcaire, pesticides, bact\u00e9ries, virus, parasites, contaminants radio-actifs…<\/li><\/ul>\n\n\n\n
